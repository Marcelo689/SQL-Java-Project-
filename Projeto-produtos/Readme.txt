Instalar o java jdk para poder usar o java. 

Instalar uma ide para utilizar o java, no meu caso irei utilizar Apache netbeans.

Instalar o xammp.

Abra o xammp e inicie os serviços apachee mysql (clicando no botão start ao lado deles).

Abra o navegador de sua preferencia e digite 'localhost/phpmyadmin' na sua url.
Crie um banco de dados com o nome javaprodutos e clique no importe e escolha o arquivo (dentro do 'projeto-produtos' bd.sql), clique em executar.

Abra o apache netbeans clique em file < Open Project e selecione onde esta o arquivo que você baixou do github, escolha o 'projeto-produtos'.

Depois pediram para instalar plugins, instale e clique com botão direito no projeto (Projeto-get-set), clique em Run, no Main Class digite 'view.Login' (sem aspas),
e abaixo clique em Browse no Working Directory e mostre o caminho do projeto-produtos.

Clique em executar no botão play verde, o nome e senha são marcelo,123.
